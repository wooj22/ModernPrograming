C++에서 "리터럴 클래스 타입"(literal class type)은 특정한 조건을 만족하는 클래스 타입을 말합니다. 
이러한 클래스는 상수 표현식에서 사용할 수 있도록 설계된 클래스로, 컴파일 타임에 값이 계산될 수 있어야 합니다. 
이는 주로 상수 표현식에서 사용되는 객체나 constexpr 함수 내에서 필요한 타입입니다.


리터럴 클래스는 다음과 같은 특성을 가집니다:

constexpr 함수와 변수: 클래스의 멤버 함수는 constexpr로 선언되어야 하며, 이를 통해 객체가 컴파일 타임에 생성될 수 있습니다.

복사 생성자와 이동 생성자: 복사 생성자와 이동 생성자는 constexpr로 정의되어야 합니다.

소멸자: 리터럴 클래스는 소멸자가 기본적으로 constexpr이어야 합니다. 그러나, 소멸자가 constexpr로 선언되지 않으면 리터럴 클래스가 될 수 없습니다.

상수 객체 생성 가능: 리터럴 클래스는 컴파일 타임에 상수 객체를 생성할 수 있어야 하며, 이러한 객체는 프로그램 실행 전에 초기화됩니다.

정적 메모리 할당: 리터럴 클래스는 동적 메모리 할당을 하지 않는 것이 좋습니다. 즉, new나 delete 연산자는 사용할 수 없습니다.



#include <iostream>

class Point {
public:
    constexpr Point(int x, int y) : x_(x), y_(y) {}

    constexpr int getX() const { return x_; }
    constexpr int getY() const { return y_; }

private:
    int x_, y_;
};

int main() {
    constexpr Point p(10, 20);  // 컴파일 타임에 객체 생성
    std::cout << "x: " << p.getX() << ", y: " << p.getY() << std::endl;
    return 0;
}
